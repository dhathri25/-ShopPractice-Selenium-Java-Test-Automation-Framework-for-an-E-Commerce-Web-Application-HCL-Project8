# E-Commerce Automation Framework

## Project Description

This project is a Selenium Java Automation Framework developed using Page Object Model (POM) for an e-commerce application. It automates end-to-end workflows like login, product selection, cart operations, checkout, and order validation.

---

## Features

- Login and Logout Automation  
- Product Listing and Selection  
- Add to Cart and Cart Validation  
- Delete Product from Cart  
- Checkout and Order Placement  
- Order History Verification  
- Form Validations (Login/Register)  
- Explicit & Fluent Wait Handling  
- Screenshot on Failure  
- Extent Reports Integration  

---

## Tech Stack

- Java  
- Selenium WebDriver  
- TestNG  
- Maven  
- ExtentReports  

---

## Reports

Extent Reports are generated in the `/reports` folder after test execution.

---

## Screenshots

Failure screenshots are automatically captured and stored in the `/screenshots` folder.

---


## Key Concepts Used

- Page Object Model (POM)  
- Explicit Wait & Fluent Wait  
- TestNG Annotations & DataProvider  
- TestNG Listener for Reporting & Screenshots  
- Reusable Utilities (ConfigReader, ScreenshotUtil)
